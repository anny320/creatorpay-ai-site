What it actually does

Six checks. One question: where's the money going?

Checks for missing payment methods, currency support, mobile friction, and subscription handling — scored 0 to 100.

Localized advice — M-Pesa for Kenya, Paystack for Nigeria, UPI for India — instead of one generic checklist.

Developer preview

Not on the Chrome Web Store yet — here's how to run it today.

CreatorPay AI is in early testing. Until it's listed in the Web Store, you can load it directly from the source files in about a minute.

01

Download the extension files

Get the extension folder from the project repo (manifest.json, content.js, and the rest, all in one folder).

02

Open chrome://extensions

Type it directly into your address bar — it's a built-in Chrome page.

03

Turn on Developer mode

The toggle is in the top-right corner of that page.

04

Click "Load unpacked"

Select the folder with manifest.json inside it. The extension installs instantly — no review wait.

05

Visit a supported platform

Shopify, Whop, Gumroad, Kajabi, Teachable, Patreon, or the Stripe dashboard — the widget appears automatically.
